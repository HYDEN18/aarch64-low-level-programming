# ARM64 Assembly Fundamentals (AArch64)

This repository documents my hands-on exploration of ARM64 (AArch64) assembly, with a focus on developing a strong low-level foundation required for embedded systems, kernel development, and performance-critical software.

## ⚡ Key Areas Covered

- Data movement: LDR/STR, memory access patterns
- Arithmetic: ADD, SUB, MUL, DIV
- Bitwise & shift operations
- Condition flags (NZCV) and conditional execution
- Control flow: branches, loops, and function calls
- Addressing modes and array traversal
- Stack management and procedure calls
- Register usage and ABI-level understanding

## 🧠 What This Demonstrates

- Strong understanding of CPU-level execution
- Ability to work close to hardware (registers, memory, stack)
- Deep knowledge of instruction-level behavior and optimization
- Foundations for debugging, reverse engineering, and performance tuning

## 🔗 Relevance to Embedded & Kernel Development

The concepts explored here directly map to:
- Context switching (used in RTOS and kernels)
- Interrupt handling and low-level control flow
- Bootloader and bare-metal programming
- Linux kernel and driver development

## 🛠️ Toolchain

- GNU Assembler (AArch64)
- GCC (arm-none-eabi / aarch64-linux-gnu)
- QEMU (for emulation)
- Raspberry Pi (target hardware)

## 🚧 Next Steps

- Bare-metal boot code for ARM Cortex-A
- Minimal kernel development for Raspberry Pi
- Scheduler and context switching implementation
